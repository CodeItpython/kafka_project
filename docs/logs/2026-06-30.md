# 2026-06-30 작업 로그

## 메시지 수정

- 내가 보낸 삭제되지 않은 메시지를 수정할 수 있도록 `PATCH /api/chat/rooms/{roomId}/messages/{messageId}` API를 추가했다.
- MongoDB `chat_messages` 도큐먼트에 `editedAt`을 저장해 프론트엔드에서 "수정됨" 상태를 표시할 수 있게 했다.
- 메시지 수정 시 Elasticsearch `chat-messages` 색인 문서도 같은 내용으로 다시 저장해 검색 결과가 수정된 내용과 맞게 했다.
- WebSocket `/topic/rooms/{roomId}`로 수정된 메시지를 broadcast해 현재 채팅방 화면이 같은 message id를 교체하도록 했다.
- React 말풍선에 인라인 수정 폼, 저장/취소 버튼, 수정됨 라벨을 추가했다.

## 사용한 개념

- Author-only command: 메시지 수정은 작성자 본인에게만 허용하고, 삭제된 메시지는 수정 대상에서 제외한다.
- Search index synchronization: MongoDB 원본 메시지를 수정한 뒤 Elasticsearch 검색 문서도 즉시 갱신해 읽기 모델을 최신 상태로 맞춘다.
- Realtime replacement: WebSocket으로 같은 message id의 최신 응답을 내려보내 프론트엔드가 기존 말풍선을 교체한다.
